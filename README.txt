Tomar en cuenta:
---Enfocado en Back-end principalmente ---
La parte de front-end pudiera no estar completamente desarrollada o bien, no 
tener una correcta estructura. Lo que me importa en este proyecto es que esta 
parte sea funcional, que se muestre la información correctamente y se pueda 
interactuar para realizar las principales funcionalidades. Este proyecto está
enfocado en el back-end; para practicar lo relacionado a Spring y demás, más
adelante la parte de front-end se desarrollará en Angular y back-end quedará 
en Spring haciendo peticiones REST.

Muchas secciones de las interfaces de usuario fueron obtenidas de internet, de 
diferentes sitios web por lo que yo pude o no haberlas desarrollado en su 
totalidad.
